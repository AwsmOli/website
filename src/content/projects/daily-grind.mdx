## Overview

Daily Grind helps small teams — families, roommates, friend groups, study groups — coordinate recurring responsibilities and reward follow-through with a shared points economy. Most task apps are built for individual productivity; this one is built around shared ownership, lightweight accountability, and a points ledger everyone can see.

## What's Built

The real screens exist: Welcome (the first-run join/create chooser), Login, Create Team, Join Team (QR + fallback code entry), Team Home (the board), Balance (points and the ledger), and Admin Settings. Task completion rewards points automatically, admins can adjust balances manually, and members can self-deduct — both require a note, and everything lands in the ledger.

Still ahead: hardening the Firestore security rules for production, per-user/per-team notification preferences, and moving point mutations behind Cloud Functions so balance updates are fully server-authoritative instead of trusting the client.

## Architecture

Vue 3 + Vite frontend with Pinia for state and Vue Router, backed entirely by Firebase — Auth, Firestore (with offline persistence for local-first reads/writes), Cloud Functions for scheduled reminders, and Cloud Messaging for push. Deployed to Firebase Hosting via GitHub Actions on every push to `main`, with PR preview channels along the way.
