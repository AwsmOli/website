## Overview

An exploration of combining computer vision and Ultra-Wideband (UWB) positioning into an affordable telemetry system for RC racing — automatic lap timing, racing line analysis, speed traces, sector comparisons, and race replay, the kind of data usually reserved for professional motorsport, without needing expensive commercial timing systems.

Rather than relying on one tracking technology, the plan combines multiple sensors so each can cover the other's weaknesses.

## Open Source

Intended as an open-source platform others can build on — whether that's a timing system for a local RC club, driver telemetry collection, or just experimenting with computer vision and embedded hardware. The long-term goal is a modular ecosystem where hardware, firmware, and software are all openly documented, so advanced RC race analytics are accessible to anyone with a 3D printer, a few ESP32 boards, and a camera.

Still concept stage — no hardware or code yet.

## Origin

This is the second life of "Axxelerate" — an RC-race timing app I wrote in 2016 (Ionic, Angular 1.6, Firebase) and listed on the app stores, back when I was first getting into frontend development. This concept picks the idea back up with a very different approach: vision + UWB instead of a phone app.
